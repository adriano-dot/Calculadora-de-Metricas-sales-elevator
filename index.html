Fórmulas do Funil
Funil Completo
// Etapas do funil Contatos = Leads × Taxa_Contato Qualificados = Contatos × Taxa_Qualif Propostas = Qualificados × 0.80 Fechados = Propostas × Taxa_Conversão // Receita Receita = Fechados × Ticket_Médio
Taxa de Conversão Global
Conv_Global = (Fechados / Leads) × 100 // Benchmark saudável: 3–8% // Acima de 10%: processo muito eficiente // Abaixo de 1%: revisar qualificação
Velocidade de Vendas
Velocidade = (Oportunidades × Conv × Ticket) / Ciclo_em_dias // Receita gerada por dia no pipeline // Quanto maior, mais eficiente o processo
Métricas SaaS / Recorrência
LTV — Customer Lifetime Value
LTV = Ticket_Médio / Churn_Mensal // Ex: R$2.500 / 5% = R$50.000 LTV // Ideal: LTV ≥ 3× CAC
MRR e ARR
MRR = Clientes_Ativos × Ticket_Médio ARR = MRR × 12 // MRR Net = MRR + Expansão - Churn MRR_Net = MRR × (1 + Upsell% - Churn%)
CAC e Payback
CAC = Custo_Total_Vendas / Novos_Clientes Payback = CAC / Ticket_Médio (em meses) // Payback ideal: < 12 meses LTV_CAC = LTV / CAC // ideal ≥ 3×
